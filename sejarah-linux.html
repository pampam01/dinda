Sejarah Sistem Operasi Linux

  • Dinda Putri Ramadani (8020220216)
  • 07 Oktober 2023
  • Tugas 1Sejarah

Linux merupakan sebuah sistem operasi computer yang diciptakan oleh seorang teknisi perangkat lunak dari Finlandia bernama Linus Benedict Torvalds pada awal tahun 1990an.Torvalds mulai membentuk sistem Linux pada saat masih bersekolah di University of Helsinki pada 25 Agustus 1991. Terinspirasi oleh program MINIX yang ditulis oleh professor Amerika-Belanda Andrew S. Tanenbaum yang digunakan untuk Pendidikan dan tidak bisa dimodifikasi ataupun didistribusikan ulang. MINIX memiliki arsitektur 16-bit dan tidak dapat diadopsi dengan baik oleh desain PC 386 Intel.

Pada tahun 1994, Torvalds merilis kernel Linux versi 1.0 dan core sistem operasi. Pada waktu yang berdekatan, seorang software developer Amerika Richard Stallman Bersama FSF menciptakan sebuah open-source bernama GNU. Kemudian Torvalds, Stallman, dan FSF menciptakan utility untuk sistem operasi. Utilitas itu kemudian ditambahkan ke kernel Linux untuk menjadi sistem yang lengkap Bernama GNU/Linux, atau secara singkat disebut Linux.

Pada tahun 1995, Linux dipindahkan ke DEC Alpha dan Sun SPARC. Memiliki platfrom yang lebih luas. Versi 2.0 kernel Linux dirilis pada tahun 1966. Kernel versi 2.0 dapat menyajikan berbagai prosesor menggunakan symmetric multiprocessing (SMP) dan menjadi alternatif berbagai perusahaan. Perusahaan besar seperti IBM, Compaq, dan Oracle mengumumkan bahwa mereka mendukung program Linux pada tahun 1988. Selain itu, para programmer juga mulai menciptakan interface pengguna berbentuk grafik KDE.

Sekelompok developer mulai mengerjakan lingkungan grafik GNOME, menggantikan KDE pada 1999. Selain itu, IBM mengumumkan sebuah proyek yang lebih luas untuk mendukung Linux dan Linux 2.2 dirilis. Pada tahun 2002, DELL mengumumkan bahwa mereka menjadi provider sistem berbasis Linux terbesar №2 di seluruh dunia dan menjadi pabrik besar pertama yang menawarkan Linux pada setiap produknya.

Versi 2.4 dirilis pada tahun 2001. Pada tahun 2002, media memberitakan bahwa “Microsoft telah membunuh Dell Linux”. Pada tahun 2003 Linux 2.6 dirilis. Pada tahun 2004, tim Xfree86 terpecah dan bergabung dengan badan standar X yang membentuk X.Org Foundation yang menyebabkan lebih cepatnya perkembangan server X untuk Linux. Proyek openSUSE memulai distribusi gratis dari komunitas Novell pada tahun 2005. OpenOffice.org mengenalkan versi 2.0 dan kemudian mendukung standar OASIS OpenDocument.

Oracle merilis distribusi Red Hat Enterprise Linux mereka sendiri pada tahun 2006. Microsoft dan Novell menumumkan kerja sama untuk interopabilitas yang lebih baik dan perlindungan paten bersama. Pada tahun 2007, Dell mulai memperjual-belikan laptop dengan program Ubuntu yang sudah diinstal sebelumnya. Linux 3.0 dirilis pada tahun 2011. Tahun 2012 server Linux memperbarui pasar Unix. Ubuntu mengklaim 22,000,000 pengguna pada tahun 2014.Linux versi 4.0 dirilis pada tahun 2015. Pada tahun 2017, Top500 pada barisan supercomputer tercepat menjalankan Linux.

Linux versi 5.0 dan 6.0 dirilis pada tahun 2019 dan 2022. Linux berkembang dengan baik karena usaha para developer. Meskipun Linux tidak se-user-friendly Microsoft Windows dan MacOS, Linux memiliki sistem yang efisien dan dapat diandalkan. Linux menjadi server yang paling banyak digunakan di Internet. Karena merupakan program open-source, Linux banyak dimodifikasi untuk berbagai macam kegunaan, seperti untuk telepon selular, supercomputer, Android, dan sistem operasi Google untuk perangkat mobile. Chrome OS, sistem operasi Google yang menggunakan Chrome juga berbasis Linux.
